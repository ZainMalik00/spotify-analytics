# spotify-analytics
A tool to view a users most listened to songs and artists

# How To Gain Access
Due to Spotify's currrent requirements for their extended quota, this website is limited to developer mode only. To add your spotify account to the list of accessible accounts, please post a comment here: https://github.com/ZainMalik00/spotify-analytics/discussions/24
